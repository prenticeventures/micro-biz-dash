# App Store Assets

This folder contains assets needed for App Store submission.

## Screenshots

Screenshots are stored in the `screenshots/` folder.

### Current Screenshots
- **Device:** iPhone 17 Pro Max (Simulator)
- **Dimensions:** 1320 x 2868 pixels (portrait)
- **Format:** PNG, RGB color space, no alpha channel
- **Status:** Ready for iPhone 6.9" Display slot in App Store Connect

### Apple Requirements (iPhone 6.9" Display)
According to [Apple's screenshot specifications](https://developer.apple.com/help/app-store-connect/reference/app-information/screenshot-specifications), the following sizes are accepted:
- 1260 x 2736 pixels (portrait)
- 1290 x 2796 pixels (portrait)
- 1320 x 2868 pixels (portrait) ✓ **We are using this size**

### File Requirements
- Format: PNG or JPEG
- Color Space: RGB (sRGB IEC61966-2.1)
- Alpha Channel: None (flattened)
- Maximum file size: 10MB per file

### Upload Instructions
1. Go to App Store Connect → Your App → Version → App Screenshots
2. Select "iPhone 6.9-inch Display" slot
3. Upload files from `screenshots/` folder
4. Files are named: `app-store-1320x2868 - [timestamp].png`
