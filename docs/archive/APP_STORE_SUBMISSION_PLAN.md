  - **SKU:** micro-biz-dash-001 (or any unique identifier)
- [ ] Click "Create"

#### Step 1.2: App Information
- [ ] **Subtitle:** Short catchy tagline (optional, 30 chars max)
- [ ] **Category:** 
  - Primary: Games ‚Üí Action
  - Secondary: Games ‚Üí Arcade (optional)
- [ ] **Content Rights:** Confirm you have rights to all content
- [ ] **Age Rating:** Complete questionnaire (likely 4+ or 9+)

### Phase 2: App Store Assets (1-2 hours)

#### Step 2.1: App Screenshots (Required!)
**You need screenshots for:**
- iPhone 6.7" Display (iPhone 15 Pro Max, 14 Pro Max, etc.) - **Required**
- iPhone 6.5" Display (iPhone 11 Pro Max, XS Max) - Optional but recommended
- iPhone 5.5" Display (iPhone 8 Plus, etc.) - Optional

**Sizes needed:**
- 6.7": 1290 x 2796 pixels (portrait)
- 6.5": 1242 x 2688 pixels (portrait)
- 5.5": 1242 x 2208 pixels (portrait)

**What to capture:**
1. Main menu screen
2. Gameplay (level 1)
3. Gameplay (mid-level action)
4. Victory celebration screen
5. Level complete screen

**How to create:**
- Use iOS Simulator: Device ‚Üí Screenshot (or Cmd+S)
- Or take screenshots on your physical device
- I can help create/edit these if needed!

#### Step 2.2: App Preview Video (Optional but Recommended)
- 15-30 second video showing gameplay
- Same sizes as screenshots
- Can be created from simulator or device

#### Step 2.3: App Description
**What to include:**
- Engaging description (up to 4000 characters)
- Key features
- What makes it unique
- Keywords (for search)

**I can help draft this!**

#### Step 2.4: Keywords
- Up to 100 characters
- Comma-separated
- Examples: "business, entrepreneur, platformer, retro, indie game"

#### Step 2.5: Support URL
- Your website: www.microbizdash.com
- Or create a simple support page

#### Step 2.6: Privacy Policy URL (May be Required)
- Required if app collects any data
- Can be a simple page on your website
- I can help create a basic privacy policy

### Phase 3: Build and Archive (30 minutes)

#### Step 3.1: Configure for App Store
- [ ] In Xcode, select "Any iOS Device" (not simulator)
- [ ] Product ‚Üí Scheme ‚Üí Edit Scheme
- [ ] Set Build Configuration to "Release"

#### Step 3.2: Archive the App
- [ ] Product ‚Üí Archive
- [ ] Wait for archive to complete (2-5 minutes)
- [ ] Organizer window opens automatically

#### Step 3.3: Upload to App Store Connect
- [ ] In Organizer, select your archive
- [ ] Click "Distribute App"
- [ ] Choose "App Store Connect"
- [ ] Follow the wizard:
  - Upload
  - Select your team
  - Wait for upload to complete (5-10 minutes)

---

### Phase 4: Submit for Review (15 minutes)

#### Step 4.1: Complete App Store Connect
- [ ] Go back to App Store Connect
- [ ] Select your app ‚Üí "1.0 Prepare for Submission"
- [ ] Upload screenshots
- [ ] Add app description
- [ ] Add keywords
- [ ] Set pricing (Free or Paid)
- [ ] Select the build you just uploaded

#### Step 4.2: Submit
- [ ] Review all information
- [ ] Click "Submit for Review"
- [ ] Answer export compliance questions (usually "No" for games)
- [ ] Submit!
