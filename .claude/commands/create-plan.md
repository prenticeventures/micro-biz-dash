# Create Plan

Create a detailed plan for implementing a feature or fixing an issue.

## Usage

When asked to create a plan, I should:
1. Break down the task into clear, actionable steps
2. Identify which files need to be created or modified
3. Consider dependencies and potential issues
4. Provide a step-by-step implementation guide
5. Include testing steps

## Output Format

- Clear task breakdown
- File-by-file changes needed
- Testing checklist
- Potential risks or considerations
