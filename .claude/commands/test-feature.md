# Test Feature

Test a specific feature or functionality.

## Usage

When asked to test a feature:
1. Identify what needs to be tested
2. Run the development server if needed
3. Check for console errors
4. Verify functionality works as expected
5. Test on both web and iOS if applicable
6. Report any issues found

## Testing Steps

- Start dev server: `npm run dev`
- Check browser console
- Test user interactions
- Verify data persistence (if using Supabase)
- Test iOS build if feature affects mobile: `npm run ios:run`
